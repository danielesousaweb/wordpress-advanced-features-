# Advanced Custom Fields (ACF)

This project uses Advanced Custom Fields (ACF) to extend WordPress into a CRM-style platform focused on managing companies, contacts, relationships, and business intelligence data.

The original production project contains additional business rules, field validations, workflows, and client-specific configurations that have been removed for confidentiality reasons.

## Field Groups

### Companies

Stores company records, classifications, products, solutions, representatives, and organizational relationships.

### Contacts

Stores contact records associated with companies, including participation history, events, surveys, departments, and relational references.

## Notes

* Field names are preserved to demonstrate the original architecture.
* Client data and production configurations have been removed.
* This repository focuses on the technical structure rather than the business data itself.
